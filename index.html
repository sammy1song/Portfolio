Weather App

The Global Weather App is a simple and intuitive web application built with React.js, enabling users to retrieve real-time weather information from any location around the globe. With its clean, user-friendly interface, users can effortlessly search for a location and get immediate results showing the current temperature, general weather description, 'feels like' temperature, humidity, and wind speed. The data is sourced from the reliable OpenWeatherMap API, ensuring accurate and up-to-date weather readings.
